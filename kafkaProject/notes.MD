Functional interfaces are interfaces with one abstract method.
- Function<T,R>
- Consumer<T>
- Supplier<T>
- Predicate<T>

### Spring Cloud Function
    -lets you write just one java function-a supplier, function or consumer-then bind it to HTTP, Kafka or Lambda by configuration alone

### Spring Cloud Stream
    - lets you write plain java functions and then bind them to messaging destinations(topics, queues) without writng any broker specific code.
